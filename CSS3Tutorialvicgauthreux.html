10 Awesome CSS3 Features to Know

1. Rounded Corners

Using the border-radius property is how you create rounded corners.

Example Code: "div { border: 2px solid; border-radius: 25px; }"

Click here for more Information.

2. CSS3 box-shadow Property

Using the box shadow is simple, if you apply the "example" code to your property you can have a shadow cast in any direction.

Symtax: box-shadow: none|h-shadow v-shadow blur spread color |inset|initial|inherit;

Example Code: div { box-shadow: 10px 10px 5px #888888; }

Click here for more Information.

3. Text-Shadow

Having a shadow set for you text is a new feature made easy in CSS3

You specify the horizontal shadow, the vertical shadow, the blur distance, and the color of the shadow

Example Code: h1 { text-shadow: 5px 5px 5px #FF0000; }

Click here for more Information.

4. Resizing Background Images Using CSS3

CSS3 allows you to set the size of the background with a simple style now.

Example Code:

style=" background: url(images/img2.jpg); background-size: 80px 60px; background-repeat: no-repeat; padding-top: 40px;"

Click here for more Information.

5. Animation

"Bind the "myfirst" animation to the "div" element. Animation duration: 5 seconds:"

Example Code: div { -webkit-animation: myfirst 5s; /* Chrome, Safari, Opera */ animation: myfirst 5s; } /* Chrome, Safari, Opera */ @-webkit-keyframes myfirst { from {background: red;} to {background: yellow;} } /* Standard syntax */ @keyframes myfirst { from {background: red;} to {background: yellow;} }

6. CSS3 word-wrap Property

This is a great feature with CSS3, making the text wrap can inprove any and every website.

Syntax: word-wrap: normal|break-word|initial|inherit;

Example Code: p.test { width: 11em; border: 3px solid #000000; word-wrap: break-word; }

7. Create Multiple Columns

With CSS3, you can create multiple columns for laying out text - like in newspapers! In this chapter you will learn about the following multiple column properties:

The column-count property specifies the number of columns an element should be divided into:

Example Code: div { -webkit-column-count: 3; /* Chrome, Safari, Opera */ -moz-column-count: 3; /* Firefox */ column-count: 3; }

Click here for more Information.

8. Linear Gradients

Creating a linear gradient is easier now with CSS3 just define at least two color stops. Color stops are the colors you want to render smooth transitions into each. You can also set a starting point and a direction (or an angle) along with the gradient effect.

Example Code: #grad { background: -webkit-linear-gradient(red, blue); /* For Safari 5.1 to 6.0 */ background: -o-linear-gradient(red, blue); /* For Opera 11.1 to 12.0 */ background: -moz-linear-gradient(red, blue); /* For Firefox 3.6 to 15 */ background: linear-gradient(red, blue); /* Standard syntax */ }

Click here for more Information.

9. Using Bold Text

CSS3 makes another way to bold your text from the style sheet.

@font-face {font-properties}

Example Code: @font-face { font-family: myFirstFont; src: url(sansation_light.woff); } div { font-family: myFirstFont; }

Click here for more Information.

10. Transition

CSS3 transitions are effects that let an element gradually change from one style to another.

The transition properties allow elements to change values over a specified duration, making it appear the property changes.

Example Code:div { width: 100px; height: 100px; background: red; -webkit-transition: width 2s; /* For Safari 3.1 to 6.0 */ transition: width 2s; } div:hover { width: 300px; }
